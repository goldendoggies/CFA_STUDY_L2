Credit Analysis Models

The Credit Valuation Adjustment is used to determine the adjustment for credit risk (Credit Spread)

  • Expected exposure (Amount that could be lost in default without considering any recovery)
  • Credit Spread (G-spread) which is Corporate minus Govt (or benchmark) bond
  • Default Risk (Probability of Default - POD)
  • Default Risk Premium (Uncertainty of timing of default)
  • Credit Risk (If default, how much is lost? - Loss Given Default LGD)
  • Recovery Rate (% recovered after default)
  • Loss severity (1-recovery rate)
  • Loss given default
  • Probability of default
  • Credit valuation adjustment
  • Expected loss
  • Present Value of the expected loss

CVA Summary

  • Calc Par >> Spot >> Forward Rates
  • Construct Interest Rate Tree using forward rates
  • Calc Bond value assuming no default
  • Calc Expected Exposures (* 1- recovery rate) = LGD
  • LGD = (POD * DFn)
  • Calc Sum for CVA and subtract from bond assuming no default (Value no Default, VND)
  • Calc YTM (using I, PMT, PV, FV) and subtract Govt (benchmark) YTM
  • Calc Credit Spread

Present value of the expected loss

  1. Loss given default (LGD) = Expected Exposure x (1 - recovery rate)

  2. Probability of Default (POD) at time T

  3. Expected Loss (at time T) = LGD X POD at time T / Divided by (1 + Risk Free Rate)^n

Expected Change for Credit Rating Migration
  1. Calculate calculated return per spread migration. ModDur x (Credit Spread at each rating - Credit Spread of change).
    • (Ex AA >> AAA = spread AA - spread AAA)
  2. Weight each expected return by the probability of each spread migration.
  3. Expected return over next year due to credit spread migration = YTM - weighted expected returns of all migrations
